# KeeReadingBot

A bot for web page sharing

## Installation

### Requirements
* Go (1.10+)

## Usage

```
make dist/bot
BOT_TOKEN=YOUR_TOKEN BOT_CHANNEL=YOUR_CHANNEL ./dist/bot
```
